# react-puzzle-engine

A game engine for building React puzzle games.

## Development

```
npm start
npm test
```

## Features

- [x] Stopwatch
- [x] Score
- [x] Lifes
- [ ] Game states (start, pause, end)
- [ ] Levels
- [ ] Utils
    - [x] Grid
    - [ ] Array utils
    - [ ] RNGs
